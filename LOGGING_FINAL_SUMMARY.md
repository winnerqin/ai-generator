## 🔧 核心实现

### 1. 日志基础设施（web_app.py 第1-60行）
```python
logging.basicConfig(
    level=logging.INFO,
    format='%(asctime)s - [%(levelname)s] - %(message)s',
    handlers=[
        logging.FileHandler('app.log', encoding='utf-8'),
        logging.StreamHandler()
    ]
)

def log_operation(operation, details=None, level='INFO')
def log_request(method, endpoint, user_id=None, params=None)
def log_response(endpoint, status, message=None)
```

### 2. API日志集成点

#### 认证系统 (4个)
- POST /login - 用户登录
- POST /register - 用户注册
- GET /logout - 用户登出
- GET /api/auth-status - 认证检查

#### 图片生成 (4个)
- POST /api/generate - 单图生成
- POST /api/batch-generate-all - 批量生成
- GET /api/batch-progress/{batch_id} - 进度查询
- GET /output/{user_id}/{filename} - 文件下载

#### 脚本分析 (1个)
- POST /api/analyze-script - 脚本分析

#### 记录管理 (3个)
- GET /api/records - 获取记录
- DELETE /api/records/{record_id} - 删除记录
- POST /api/batch-delete - 批量删除

#### 样本管理 (3个)
- GET /api/sample-images - 获取样本
- POST /api/upload-sample-image - 上传样本
- POST /api/delete-sample-image - 删除样本

#### 库资源管理 (3个)
- POST /api/add-to-person-library - 添加人物库
- POST /api/add-to-scene-library - 添加场景库
- POST /api/delete-library-asset - 删除库资源

#### 系统启动 (1个)
- Main - 应用启动初始化

### 3. 日志内容规范

#### 操作日志格式
```
[操作] 操作名称 | 用户ID: X, 操作详情: Y
```

#### 请求日志格式
```
[请求] METHOD /endpoint | 用户: X | 参数: Y
```

#### 日志级别
- INFO：正常操作成功
- WARNING：警告、部分失败
- ERROR：错误、完全失败

## 🚀 使用流程

### 第一次使用
```powershell
# 1. 启动应用
python web_app.py

# 2. 在新终端监控日志
python log_monitor.py --watch

# 3. 使用应用，观察日志输出
```

### 日常监控
```powershell
# 实时查看日志
python log_monitor.py --watch

# 或查看统计
python log_monitor.py --stats

# 或搜索问题
python log_monitor.py --search "错误"
```

### 故障排查
```powershell
# 查看特定用户的操作
python log_monitor.py --user 123

# 查看所有错误
python log_monitor.py --errors

# 搜索关键词
python log_monitor.py --search "API超时"
```
